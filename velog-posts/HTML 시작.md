<h2 id="html-알아보기">HTML 알아보기</h2>
<br />

<p>HTML = HyperText Markup Language</p>
<ul>
<li>HyperText : 하이퍼링크를 통해 어떤 문서에서 다른 문서로 접근 가능한 텍스트</li>
<li>Markup : 콘텐츠를 표시하는 것</li>
<li>Language : 언어</li>
</ul>
<p>즉, HTML은 &quot;하이퍼 텍스트와 콘텐츠를 표시해주는 언어&quot;를 뜻합니다.</p>
<p>더 쉽게 표현해보면 웹브라우저(크롬, 엣지 등)를 통해 우리가 볼 수 있는 웹페이지의 콘텐츠들을 정의할 때 쓰이는 언어입니다.
<br /></p>
<p>개발자는 이 HTML을 통해 웹페이지에 어떤 내용을 어떻게 나타나도록 할지를 정의하는 코드를 구현합니다.
<br /></p>
<p><img alt="" src="https://velog.velcdn.com/images/lina6310/post/aa2e384b-7d0d-4a16-87e6-dc6ba89e88de/image.png" /></p>
<p>위와 같은 순서로 HTML 코드가 웹페이지에서 표시됩니다.
이렇게 HTML 코드가 웹브라우저를 통해 해석되고 표현되는 과정을 <strong>&quot;렌더링&quot;</strong>이라고 합니다.
<br /></p>
<p>HTML 문서의 확장자 = .html 또는 .htm
<br />
<br /></p>
