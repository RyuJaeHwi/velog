<h2 id="css-알아보기">CSS 알아보기</h2>
<p>CSS = Cascading Style Sheets</p>
<ul>
<li>Cascading : 계단식</li>
<li>Style : 스타일</li>
<li>Sheets : 문서 한 장</li>
</ul>
<p>즉, CSS는 &quot;계단식으로 스타일 규칙들이 적용될 수 있도록 정의하는 스타일 문서&quot;를 뜻합니다!
<br /></p>
<blockquote>
<p><strong>CSS란?</strong>
: 웹 페이지의 디자인과 레이아웃 정의에 사용되는 스타일시트 언어</p>
</blockquote>
<p>CSS는 .html 문서에 스타일, 즉 디자인을 더해줍니다.
<br /></p>
<p align="center">
  <img src="https://velog.velcdn.com/images/choco_dev/post/0f6f0a92-0da0-4e40-9cf6-8015690051c2/image.png" width="80%" />
</p>

<p>정리하면 다음과 같습니다.</p>
<ul>
<li><strong>HTML</strong> : 웹 문서의 콘텐츠를 구성하는 언어</li>
<li><strong>CSS</strong> : 웹 문서의 콘텐츠에 스타일을 추가하는 언어 (색상, 크기, 위치 등)</li>
</ul>
<br />

<p>CSS 문서(코드) 자체는 독립적으로 존재할 수 있지만, CSS 자체가 .html 문서의 스타일을 정의하는 것이니 따로 존재해도 의미가 없습니다.</p>
<p>그러니까 .html 문서와 함께 있는 것이 아니라면, CSS 문서는 단순한 텍스트가 됩니다.</p>
